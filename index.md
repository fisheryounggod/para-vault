# PARA Vault

欢迎来到我的Obsidian PARA知识管理系统。

## 关于 PARA 方法

PARA 是一种高效的知识管理方法，由 Tiago Forte 提出：

- **Projects (项目)**: 当前正在进行的任务，有明确的时间表
- **Areas (领域)**: 长期责任和关注点
- **Resources (资源)**: 未来可能用到的主题和兴趣
- **Archives (归档)**: 已完成的项目和不再活跃的内容
